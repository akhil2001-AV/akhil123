A grid

is a collection of horizontal and vertical lines creating a pattern against which we can line up our design elements. They help us to create layouts in which our elements won't jump around or change width as we move from page to page, providing greater consistency on our websites. Grid is merely a skeleton that helps arrange the items on your page, so use it wisely. Four types of grid layouts can help you establish a well-balanced landing page. Use Block, Multicolumn, Modular, and Baseline grids to create a visual hierarchy on your page, and you are guaranteed to increase your conversions.

Types of Grid

  • Baseline Grid
  • Column Grid
  • Modular Grid
  • Manuscript Grid
  • Pixel Grid
